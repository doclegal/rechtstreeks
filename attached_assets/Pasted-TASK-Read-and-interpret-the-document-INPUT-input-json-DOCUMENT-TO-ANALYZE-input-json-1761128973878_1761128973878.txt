TASK

Read and interpret {{input_json}} provided.
Determine if the document is readable and complete.

Identify the document type (e.g., factuur, contract, bewijs van betaling, correspondentie, rapport, foto, etc.).

Evaluate whether the document logically belongs to the case, based on:

The case title and description

  "is_readable": true,
  "belongs_to_case": true,
  "summary": "[1–2 sentence summary in Dutch]",

  "note": null
}

RULES

Summary: Always in Dutch, max. 2 sentences.

Tags: Dutch words, 3–6 per document.

Note: Only fill in when:

Document seems unrelated to the case

Document is unreadable or corrupted

Document date is clearly outside the relevant timeframe

Content seems incomplete or suspicious

If everything seems fine → note = null.

Do not be overly cautious — if the document appears relevant, set belongs_to_case = true.
