Init & clients

from pinecone import Pinecone
import os
pc = Pinecone(api_key=os.environ["PINECONE_API_KEY"])
index = pc.Index("jouw-index-naam")  # de naam uit de console


Upsert text (tekst + metadata)

rows = [
  {"id": "ECLI:NL:RBAMS:2024:1234#001",
   "values": None,  # bij integrated embedding laat je values weg
   "metadata": {"ecli":"ECLI:NL:RBAMS:2024:1234","domain":"huurrecht","url":"https://..."},
   "text": "…chunk_text van de uitspraak…"}
]
index.records.upsert(namespace="default", records=rows)


(Bij integrated embedding converteert Pinecone text → vectors zelf.)

Node/JavaScript (backend)

SDK

npm i @pinecone-database/pinecone


Init

import { Pinecone } from "@pinecone-database/pinecone";
const pc = new Pinecone({ apiKey: process.env.PINECONE_API_KEY });
const index = pc.index("jouw-index-naam");


Upsert text + metadata

await index.namespace("default").upsert([
  {
    id: "ECLI:NL:RBAMS:2024:1234#001",
    metadata: { ecli:"ECLI:NL:RBAMS:2024:1234", domain:"huurrecht", url:"https://..." },
    text: "…chunk_text…"
  }
]);


Search + filter

const res = await index.namespace("default").search({
  query: { text: "ontbinding huurovereenkomst wegens overlast" },
  filter: { domain: { $eq: "huurrecht" } },
  topK: 10
});
